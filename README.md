# Simple DC Tool - by Trippy  v2.0

## Features
- **Accounts** — save accounts, validate tokens, view servers, add notes, copy token
- **Server Backup** — full backup of roles/channels/emojis/stickers, compare two backups, restore preview
- **Server Info** — lookup any server by invite link or ID
- **User Lookup** — get public profile, account creation date, badges, avatar
- **Invite Tracker** — list all invites in a server with use counts
- **Permission Checker** — see exactly what permissions your bot has
- **Webhook Sender** — send messages + embeds via webhook URL
- **Settings** — accent color picker, default bot token pre-fill

---

## Getting tokens

### User Token
1. Open Discord in browser (discord.com/app)
2. Press F12 → Network tab
3. Send any message → find the request → look for `Authorization` header

### Bot Token
1. Go to https://discord.com/developers/applications
2. New Application → Bot → Reset Token → Copy
3. Invite bot to your server with the permissions you need

### Guild ID
1. Enable Developer Mode in Discord settings
2. Right-click your server → Copy Server ID
